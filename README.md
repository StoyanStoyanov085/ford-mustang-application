An application designed specifically for fans of the American brand Ford Mustang


Tech Stack
List of technologies used in the application:

- Frontend: HTML/CSS
- Backend:  Java Spring
- Database: MySQL
        

Supported Features
The key features supported by the application:

- User registration and authentication
- Role-based access control
- Data CRUD operations
- Real-time data updates


Functionalities
Detailed overview of application functionality:

- Data validation during user registration
- Data validation during user login 
- Creating cars
- CRUD operations cars 
- CRUD operations user/admin
- Implements business logic in modular service layers
- Error Handling logging and exception management
